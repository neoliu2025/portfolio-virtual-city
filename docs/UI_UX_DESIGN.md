## 1. 项目概述

### 1.1 项目简介
一个基于 2.5D 等距视角的虚拟城市作品展示平台，用户通过探索城市的不同区域来了解设计师的信息、作品、成长经历等。

### 1.2 核心价值主张
- **沉浸式体验**: 打破传统作品集展示方式，进入虚拟城市探索
- **交互高粘性**: WASD 移动 + 点击探索，增加用户停留时间
- **多维度展示**: 同一空间中展示多种内容维度
- **记忆点强**: 城市环境比列表更容易记住

### 1.3 目标用户
- 设计师/开发者的访客
- 潜在客户或合作方
- 社交媒体分享者（高分享欲望）

### 1.4 成功指标
- 平均停留时间 > 3 分钟
- 移动端完成度 > 80%
- 社交分享率 > 15%

## 5. 交互设计详解

### 5.1 移动控制系统

#### 桌面控制
```
WASD键:
  W - 向上移动视角 (北)
  A - 向左移动视角 (西)
  S - 向下移动视角 (南)
  D - 向右移动视角 (东)

鼠标:
  移动 - 旋转摄像机角度
  点击 - 选择/进入
  滚轮 - 缩放视图
  右键拖拽 - 平移视图

快捷键:
  Q - 打开快速菜单
  M - 显示/隐藏小地图
  ESC - 返回/关闭菜单
  Space - 跳跃/动作确认
  R - 重新生成城市 (彩蛋)
  T - 切换时间 (日/夜模式)
```

### 5.2 动画设计规范

#### 过渡动画
```
页面加载进入:
  时长: 1.5s
  效果: 淡入 + 缓动
  
建筑进入:
  时长: 0.8s
  效果: 放大 + 旋转 + 淡入
  
建筑退出:
  时长: 0.6s
  效果: 缩小 + 淡出
  
摄像机移动:
  时长: 1.2s - 2s (基于距离)
  效果: easeInOutQuad 缓动
```

#### 微动画
```
按钮悬停:
  时长: 0.2s
  效果: 缩放 1 → 1.1 + 阴影加深

建筑闪烁:
  时长: 0.6s (无限循环)
  效果: 不透明度 0.8 → 1.0 → 0.8

文字逐字显示:
  延迟: 50ms per char
  效果: 从左到右淡入

粒子效果:
  时长: 2-3s
  效果: 升起 + 淡出
```

## 6. 视觉设计系统

### 6.1 字体系统

```
无配色字体规范:

标题字体:
  - 字族: 现代无衬线 (如 Poppins, Inter)
  - 用途: 主标题, 建筑名称
  - 大小: 28px - 48px
  - 粗细: Bold (700)

正文字体:
  - 字族: 清晰无衬线 (如 Open Sans)
  - 用途: 内容, 描述
  - 大小: 14px - 18px
  - 粗细: Regular (400)

辅助字体:
  - 字族: 等宽字体 (如 Monaco)
  - 用途: 代码, 时间戳
  - 大小: 12px - 14px
  - 粗细: Regular (400)
```

### 6.2 布局网格

```
基础网格: 8px 系统
  - 间距单位: 8px 的倍数
  - 8px, 16px, 24px, 32px, 40px...

响应式断点:
  Mobile: < 768px
  Tablet: 768px - 1024px
  Desktop: > 1024px
```

## 7. 性能与优化策略

### 7.1 加载优化

```
资源优先级:
  1. HTML + CSS + JS (关键)
  2. 城市基础模型 (高)
  3. 背景音乐 (中)
  4. 建筑细节 (低)
  5. 动画资源 (低)

压缩策略:
  - 3D模型: GLTF + Draco 压缩
  - 纹理: WebP 格式 + 尺寸优化
  - 音频: MP3/AAC 格式 + 128kbps
  - 代码: 压缩 + Tree-shaking
```

### 7.2 渲染优化

```
性能目标:
  - FPS: 60fps (PC), 30fps (Mobile)
  - 首屏: < 3s
  - 交互响应: < 100ms

优化手段:
  - LOD (细节层级)
  - 视椎体剔除
  - 对象池 (缓存)
  - 批量渲染
```
